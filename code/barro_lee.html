Let’s use the LASSO penalized method in order to select the most important variables.

X <- growth %>% 
    dplyr::select(-Outcome, -intercept) %>% 
    as.matrix

y <- growth %>% 
    pull(Outcome)

There is something called \(\lambda\). But we did not specify any value for lambda.

length(lasso.fit$lambda)
## [1] 100

That is, even when we do not specify the values for lambda, the glmnet() will provide some values by default.

plot(lasso.fit, label = T)

In order to choose the best value for \(\lambda\), we use the cross validation. glmnet package offers a handy function to do that called cv.glmnet():

cv.fit <- cv.glmnet(x = X,
                    y = y, 
                    nfolds = nrow(X)-1, 
                    standardize = T)
## Warning: Option grouped=FALSE enforced in cv.glmnet, since < 3 observations
## per fold
